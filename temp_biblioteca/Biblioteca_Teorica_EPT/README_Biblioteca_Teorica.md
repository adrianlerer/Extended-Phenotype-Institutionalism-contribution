# README: BIBLIOTECA TEÓRICA EPT

## Propósito

Esta biblioteca contiene la fundamentación teórica completa para posicionar **Extended Phenotype Theory (EPT)** como nuevo paradigma en teoría general del derecho, siguiendo la estructura de revoluciones científicas de Thomas Kuhn.

---

- **Conceptos clave:** Fenotipo extendido, memes, fitness landscape, construcción de nichos, cooperación

- **Conceptos clave:** Paradigma, revolución científica (6 fases), anomalías, crisis, inconmensurabilidad, resistencia generacional
- **Aplicación EPT:** Epistemología - ¿CÓMO cambian paradigmas? → EPT es revolución kuhniana
- **Citas textuales:** 8 citas utilizables directamente en papers
- **Ejemplos históricos:** 5 revoluciones (Copérnico, Newton, Lavoisier, Einstein, Maxwell) como plantillas
- **Estructura:** Arquitectura completa del cambio paradigmático, síntomas de crisis, condiciones para nuevo paradigma

#### **DENNETT_Peligrosa_Idea_Darwin_Ficha_Bibliografica.md** (41KB)
- **Contenido:** Análisis exhaustivo de "La Peligrosa Idea de Darwin" (1995)
- **Conceptos clave:** Algoritmo darwiniano, grúas vs ganchos celestes, diseño sin diseñador, memes, espacio de diseño, peligrosidad, ácido universal
- **Aplicación EPT:** Mecanismo - ¿CÓMO evolucionan instituciones? → Algorítmicamente (variación + herencia + selección)
- **Citas textuales:** 8 citas utilizables directamente en papers
- **Estructura:** Algoritmo universal, crítica a ganchos celestes, neutralidad del sustrato, carrera de armamentos memética

## Cómo Usar Esta Biblioteca

### Para escribir un paper defendiendo EPT:

1. **Empiece con el ÍNDICE MAESTRO** (INDICE_MAESTRO_Biblioteca_Teorica_EPT.md)
   - Vaya a "Sección VII. Guía de uso para papers"
   - Elija estructura sugerida según tipo de paper (algoritmo darwiniano, revolución kuhniana, metodología EPT)

2. **Identifique conceptos necesarios:**
   - Use "Sección I. Índice por concepto clave" para encontrar definiciones precisas
   - Use "Sección IV. Índice de citas textuales por tema" para encontrar citas específicas

3. **Consulte fichas individuales para profundizar:**
   - Para ontología (qué son instituciones) → DAWKINS
   - Para epistemología (cómo cambian paradigmas) → KUHN
   - Para mecanismo (cómo evolucionan) → DENNETT

---

### Para responder a críticas:

1. **Vaya al ÍNDICE MAESTRO → Sección V. Índice de argumentos contra críticas**
2. **Identifique la crítica específica** (hay 7 críticas anticipadas)
3. **Use la respuesta estructurada** basada en Dawkins/Kuhn/Dennett
4. **Consulte ficha individual** si necesita más profundidad

**Ejemplo:**
- Crítica: "EPT justifica desigualdad/injusticia"
- Respuesta: Índice Maestro → Sección V → Crítica 1 → Dennett (falacia naturalista)
- Profundizar: DENNETT ficha → Críticas anticipadas #1

---

## Conceptos Clave Integrados

### Triángulo conceptual fundamental:

```
       DAWKINS (Ontología)
            ↓
   ¿Qué son instituciones?
   Fenotipos extendidos de memes
            ↓
            ↓
       DENNETT (Mecanismo) ←→ KUHN (Epistemología)
            ↓                      ↓
  ¿Cómo evolucionan?         ¿Cómo cambian paradigmas?
  Algorítmicamente           Revolución kuhniana
  (variación+herencia        (anomalías→crisis→
   +selección)                revolución)
```

### Síntesis EPT (integrando los 3 autores):

1. **Ontología (Dawkins):** Instituciones legales = fenotipos extendidos de memes cooperativos
2. **Mecanismo (Dennett):** Memes evolucionan algorítmicamente (variación, herencia, selección)
3. **Epistemología (Kuhn):** EPT emerge como revolución científica tras crisis de positivismo/natural law
4. **Validación (Paper SSRN):** IusMorfos + CLI operacionalizan conceptos teóricos; R² = 0.76 valida predicciones
