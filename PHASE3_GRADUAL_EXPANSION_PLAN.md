## Quality Control Protocol

### Coding Standards (All Phases)

**1. Time Investment**
- **Minimum**: 2.5 hours per case
- **Target**: 3 hours per case
- **Never rush**: Quality over deadline adherence

**2. Evidence Requirements**
Each case must document:
- ‚úÖ CSI Component 1 (Citation Endogamy): Quantitative citation analysis OR expert assessment
- ‚úÖ CSI Component 2 (Sacralization): Examples of untouchable concepts with evidence
- ‚úÖ CSI Component 3 (Costly Signaling): Documented exclusion mechanisms
- ‚úÖ CSI Component 4 (Institutional Control): Mapping of gatekeeper institutions
- ‚úÖ REI Component 1 (Implementation Rate): Percentage data from official sources
- ‚úÖ REI Component 2 (Outcome Alignment): Comparison of stated vs achieved objectives
- ‚úÖ REI Component 3 (Adaptability): Evidence of adjustments based on feedback
- ‚úÖ Control Variables: Legal family, region, GDP, population
- ‚úÖ Sources: Minimum 3 independent sources per case

**3. Coding Notes Requirement**
- Each batch must have accompanying `.md` file documenting:
  - Sources consulted
  - Measurement decisions
  - Uncertainty/limitations
  - Alternative interpretations considered

**4. Data Sources Priority**
- **Tier 1 (Preferred)**: Primary legal documents, official statistics, court databases
- **Tier 2 (Acceptable)**: Peer-reviewed academic studies, OECD/World Bank reports
- **Tier 3 (Use sparingly)**: Quality journalism, NGO reports, expert interviews
- **Never use**: Blog posts, social media, unsourced claims

## Risk Management

### Potential Risks & Mitigation

**Risk 1: Correlation weakens with more diverse sample**
- **Mitigation**: This is actually a GOOD scientific outcome - we learn boundaries
- **Response**: Investigate moderators (e.g., "Effect only in democracies?")

**Risk 2: Time investment exceeds estimate**
- **Mitigation**: Build in 20% buffer (360 hours instead of 300)
- **Response**: Adjust timeline, never sacrifice quality for speed

**Risk 3: Data availability issues in some jurisdictions**
- **Mitigation**: Have backup jurisdiction list ready
- **Response**: Substitute with similar jurisdictions from same region

**Risk 4: Coding fatigue / quality degradation**
- **Mitigation**: Take breaks between batches, re-code random 10% for quality check
- **Response**: If quality drops, pause and refresh methodology

**Risk 5: New findings challenge original theory**
- **Mitigation**: This is GOOD - science should be falsifiable
- **Response**: Update theory, don't manipulate data to fit predictions
